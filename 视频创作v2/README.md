# 视频创作v2·分镜 Prompt 总览

> v2 在 [[../视频创作/README]] 基础上重构,核心改动两条:
> 1. **单段上限从 10s 提到 15s**(适配 Sora / 可灵 1.6 长片段模式)
> 2. **段间冗余 + `@` 引用** —— 关键段(段 03)生成两次,第二轮用第一轮输出的图片作 i2v 锚点
>
> 最终交付:**5 段分镜 × 2 轮生成** = 6 段 prompt 文件 = ~65s 完整视频

## 🎯 v1 → v2 关键差异

| 维度 | v1(9 段,10s 上限) | v2(5 段,15s 上限) |
|------|:---:|:---:|
| 段数 | 9 | 5(段 03 拆 03a/03b,共 6 文件) |
| 单段上限 | 10s | 15s |
| 单段最长 | 10s(段 08 临界) | 15s(段 05) |
| 单段最短 | 4s(段 09) | 12s(段 01) |
| 总时长 | 64s | 67s(段内冗余 4-5s 重复) |
| 角色一致性手段 | 每段重写 CHARACTER ANCHOR | 每段用 `@温柚` 角色图 + 上一段 `@图片` 作 i2v |
| 跨段衔接手段 | 剪映 Dissolve 转场 | **段间冗余**(相邻段头尾重叠 2-3s) + 二次生成 |

## 📐 5 段分镜总览(总时长 ~65s)

| 段 | 文件 | 时长 | 段落大意 | 轮次位置 |
|:---:|------|:---:|----------|:---:|
| **01** | [[01_段01_晨·卧室全景+温柚出场]] | 12s | 卧室全景空镜 → 温柚半靠枕头 → 猫耳晃动特写 | R1·1 |
| **02** | [[02_段02_晨·睡眼+脚踝+好友+揉耳]] | 13s | 睡眼近景 → 脚踝蜷缩 → 好友推门 → 揉耳互动 | R1·2 |
| **03a** | [[03a_段03_晨午·喝牛奶+换装+街道]] | 15s | 喝牛奶 → 掀被换装 → 街道并肩 → 花店接花起始 | R1·3 |
| **03b** | [[03b_段03_午·街道+花店+抱花]] | 15s | 街道并肩(续)→ 花店接花轻嗅 → 抱花特写 → 抱花走在街道 | R2·1 |
| **04** | [[04_段04_午·甜品橱窗+吃舒芙蕾]] | 13s | 抱花走在街道(续)→ 橱窗央求 → 吃舒芙蕾擦奶油 | R2·2 |
| **05** | [[05_段05_午后夕·插花+蜜桃+窗边落日]] | 15s | 归家插花翻书 → 吃蜜桃 → 窗边夕阳 → 落日空镜 | R2·3 |

> **段 03 双文件说明**:`03a` 在 R1 中作为"晨→午过渡",`03b` 在 R2 中作为"午段开场"。
> 两者覆盖**同一组镜头**(镜 08~12),但 prompt 强调点不同,且 `03b` 显式引用 `03a` 输出的图片作为 i2v 起始帧,确保角色/场景/光影完全一致。

## 🔁 段间冗余(关键设计)

> ⚠️ 这是 v2 的核心改动。每两个相邻段在边界处**共享 2-3 秒画面**,剪辑时只需硬切,无需 Dissolve。

```
                            │ 喝牛奶 11-13 ◀─┼───────┼▶ 喝牛奶 0-3    │
```


**冗余总时长 ~10s**(分布在 4 个衔接点,每处 2-3s)。剪映/PR 里直接硬切,不需要任何转场滤镜。

## 📌 @ 引用体系

> 每段 prompt 头部声明本段用到的 `@资源`,格式:`@<名称> = <本段引用目的>`
> AI 平台(如可灵 1.6 / Sora / Runway Gen-3)支持上传多张参考图,prompt 里写 `@温柚` 即指代已上传的角色图。

### 角色锚点(全局可用,每个段都用)

| @ 名称 | 来源 | 用途 |
|--------|------|------|
| `@温柚` | 角色人设图([[../第一步：txt2img ，img2img]] 生成) | 锁定三色长发、琥珀眼、猫耳、米杏毛衣 |

### 场景锚点(段间传递,随生成进度累积)

| @ 名称 | 首次出现 | 后续使用段 | 用途 |
|--------|:---:|:---:|------|
| `@卧室空镜` | 段 01 | 段 02 | 保持卧室场景一致 |
| `@温柚半靠` | 段 01 | 段 02(睡眼延续) | 角色姿态锚点 |
| `@揉耳互动` | 段 02 | — | 互动场景参考 |
| `@街道场景` | 段 03a | **段 03b** | **核心:i2v 起始帧** |
| `@花店场景` | 段 03a | **段 03b** | **核心:i2v 起始帧** |
| `@花束特写` | 段 03b | 段 04(抱花走)、段 05(归家插花) | 主线道具锚点 |
| `@甜品店` | 段 04 | 段 05(归家前一刻) | 场景一致 |
| `@窗边夕阳` | 段 05 | — | 收尾画面 |

> 💡 关键技巧:**段 03b 必须用 03a 输出的尾帧作 i2v 输入**(用 `@街道场景` + `@花店场景` 引用),
> 这样第二轮生成的"街道→花店"画面在角色外形、衣着、光影上与第一轮**完全对齐**。

## 🎞 生成工作流(对应两轮 6 次请求)

```
┌─ Round 1: 3 次生成 ────────────────────────────────┐
│  ① 上传角色图 @温柚                               │
│  ② 段 01 prompt → 选 12s → 输出 12s 卧室晨景视频  │
│  ③ 从输出中截取最后一帧 → 存为 @温柚半靠(睡眼)    │
│  ④ 段 02 prompt(+@卧室空镜 +@温柚半靠)→ 13s      │
│  ⑤ 从输出截取最后一帧 → 存为 @揉耳互动           │
│  ⑥ 段 03a prompt(+@揉耳互动)→ 15s                 │
│  ⑦ 从输出截取 11-15s 范围的尾帧 → 存为            │
│     @街道场景 + @花店场景                          │
└─────────────────────────────────────────────────────┘
                        ↓
┌─ Round 2: 3 次生成 ────────────────────────────────┐
│  ⑧ 段 03b prompt(+@温柚 +@街道场景 +@花店场景)    │
│     → 选 i2v 模式 → 15s(与 03a 内容近似但独立生成) │
│  ⑨ 从输出截取抱花特写 → 存为 @花束特写            │
│  ⑩ 段 04 prompt(+@花束特写)→ 13s                  │
│  ⑪ 段 05 prompt(+@花束特写 + 段 04 尾帧)→ 15s     │
└─────────────────────────────────────────────────────┘
                        ↓
            剪映/PR 导入 5 段(03a 选优 / 03b 选优)
                        ↓
           硬切拼接(冗余 2-3s 重复,只保留一份)
                        ↓
          套暖色滤镜 + 颗粒 + 柔光,BGM 一首钢琴曲
                        ↓
              输出 MP4 / 1080p / 24fps
```

> **关于 03a vs 03b 选优**:两版内容覆盖同一组镜头(镜 08~12),通常选**画面更稳、表情更自然**的那版作为最终段 03。
> 备选版可作为镜头组接续的素材库(挑几帧插入段 04 开头)。

## 🔁 一致性铁律(每段 prompt 必带)

> ⚠️ **每段 prompt 头部都必须包含「角色简述」**:
>
> ```
> @温柚: 18yo half-cat-girl, tri-color hair (black/white/orange),
> amber eyes, fluffy tri-color ears+tail, off-shoulder cream-apricot
> chunky-knit sweater, painterly anime style, soft warm light.
> ```
>
> 简述和 `@温柚` 角色图**同时使用**:图锁形象,文字锁调性。少一个就跳镜。

## ⚠️ 重要约束(从 [[../04 视频分镜脚本_90秒版]] 继承)

- **光影时段不可错位**:晨光(暖金)→ 午后(暖黄)→ 夕阳(橘粉),**绝对不可逆序**
- **主线道具**:洋甘菊花束在 03a/b 购入 → 段 04 带着逛街 → 段 05 插瓶,**任何段都不能丢**
- **调性**:慢、柔、轻、慵懒、治愈,所有运镜不超过 0.5x 速
- **角色**:温柚 + 好友(短发女性,无兽耳)
