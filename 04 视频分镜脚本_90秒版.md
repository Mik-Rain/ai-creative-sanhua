# 暖阳与三花猫少女 视频分镜脚本(90 秒版)

> 基于 [[03 分镜脚本01]] 优化:
> - 原 19 镜共 62 秒,扩展为 19 镜共 **90 秒(一分半)**
> - 增加过渡空镜与留白镜头,保持治愈慵懒调性
> - 不引入新人物/新场景,故事连贯性与原文完全一致

## 整体设定

- **画面风格**:二次元写实厚涂风,治愈系日常向,低对比度柔光质感
- **核心色调**:暖金、米白、浅橘为主,黄昏段过渡到橘粉;全程暖调光影
- **节奏调性**:舒缓慵懒,运镜平缓,空镜留白 ≥ 1 秒
- **统一人设**:三花猫耳少女温柚——黑白橘三色渐变长发、琥珀色眼眸、米杏色系针织衣物、蓬松三花尾巴;邻居好友(短发女性,无兽耳特征)
- **统一场景资产**:卧室大床、米白纱帘、白色陶瓷花瓶、玻璃牛奶杯、甜品店、街道花店
- **镜头比例**:16:9,24fps,横屏

## 段落结构

```
晨 01~09  ──  卧室起床 + 整理衣物          (43s, 47.8%)
午 10~14  ──  街道 + 花店 + 甜品店        (24s, 26.7%)
午后 15~17 ──  归家插花 + 闲坐 + 吃蜜桃    (12s, 13.3%)
夕 18~19  ──  窗边 + 落日收尾             (11s, 12.2%)
```

## 关键连续性约束(防止 AI 跳镜失忆)

1. **温柚人设恒定** — 猫耳位置、尾巴花纹(黑/白/橘三块)、米杏色毛衣、黑色长发挑染橘色这三处最容易跑偏,**每个镜头 prompt 都必须重申**。
2. **洋甘菊花束是主线道具** — 镜 11/12 购入 → 镜 15 插瓶 → 镜 18 再次抱起,**绝对不能消失**。
3. **光影时段不可错位** — 晨光偏金白、午后偏暖黄、夕阳偏橘粉;**禁止出现逆序**。
4. **声音设计(可选)** — 全程铺一层低频环境白噪 + 远处的鸟鸣(晨)、风铃(午后)、虫鸣(夕),配 BGM 一首治愈钢琴曲即可。

## 与原始脚本的差异说明

| 改动 | 原因 |
|------|------|
| 镜 02 由 4s → 6s | 角色首次出场,需要多给 1-2 秒建立形象 |
| 镜 07 由 4s → 6s | 互动是核心情绪点,延长能强化"黏人"感 |
| 镜 09 由 4s → 5s | 增加换装完整动作的呈现 |
| 镜 15 由 4s → 5s | 午后段落承上启下,需要节奏缓冲 |
| 镜 18 由 5s → 6s | 收尾前最重要的高光,延长可强化余韵 |
| **总时长由 62s → 90s** | 满足"一分半"目标,且不引入新情节 |
