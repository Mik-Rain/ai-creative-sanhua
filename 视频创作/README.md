# 视频创作·分段 Prompt 总览

> 本目录存放**每次发给 AI 创作平台的 prompt**,**1 个 .md 文件 = 1 次生成请求**。
> 因为平台单次生成上限 **10s**,所以原 62 秒分镜([[../03 分镜脚本01]])被切分为 **9 段**,每段 4-9 秒。
> 把这 9 段按顺序拼接 = 完整 ~62 秒视频(接近用户目标"60 秒左右")。

## 🎯 切段核心原则

1. **不破坏故事连续性** — 相邻短镜合并,同场景内不切(如镜 01/02/03 都是卧室晨光,合并)
2. **不超 10s** — 每段控制在 4-9s,段 08 临界 10s(需平台允许)
3. **不切关键动作** — 完整动作保留在同一段(喝牛奶、吃舒芙蕾不切开)
4. **情绪段落对齐** — 晨/午/午后/夕 四个大段落,段号不跨时段(段 08 是 午后→夕 的临界段,内部顺序衔接 OK)

## 🔁 一致性铁律(每个 prompt 都要带)

> ⚠️ **每段 prompt 头部都必须包含「CHARACTER ANCHOR」**:
>
> ```
> Anime catgirl "Wen You", 18yo, half-human half-cat.
> Hair: waist-length tri-color (black, white, orange) loose waves, messy.
> Eyes: amber, sleepy/gentle.
> Cat ears: tri-color fluffy, twitch naturally.
> Tail: long fluffy tri-color, moves naturally.
> Outfit: oversized off-shoulder cream-apricot chunky-knit sweater,
>         later paired with cream knit long skirt.
> Style: semi-realistic anime, painterly, soft warm light (gold/cream/peach).
> ```
>
> **少一句就跳镜失忆** —— 三色毛发分布、猫耳位置、米杏色毛衣这三处最容易跑偏。

## 🎞 后期拼接流程

```
9 段生成完毕
   ↓
剪映 / CapCut / Premiere Pro 导入 9 个片段
   ↓
按 01→09 顺序排列
   ↓
每两段之间加 0.5s 溶解(Dissolve)转场
   ↓
全程叠加:"暖色 +10, 颗粒 5, 柔光 +15" 滤镜
   ↓
音轨 1:BGM 一首治愈钢琴曲(80-90 BPM)
音轨 2:环境白噪(晨鸟鸣→午后风铃→傍晚虫鸣),全程 -20dB
   ↓
片头 1.5s 黑场 + 标题"暖阳与三花猫少女"(可选)
片尾 2s 黑场 + 淡出
   ↓
导出 MP4 / H.264 / 1080p / 24fps
```

## ⚠️ 跨段衔接检查清单

- [ ] 段 1→2:温柚的**半靠姿势**和**三花毛衣**保持一致
- [ ] 段 2→3:玻璃杯从无→有(段 3 第一次出现)
- [ ] 段 3→4:尾巴从被窝出来(段 3)→下床(段 4)需衔接自然
- [ ] 段 4→5:**场景切换** 街道 → 花店,需要明确室内外变化
- [ ] 段 5→6:洋甘菊从**抱着** → **放下**(段 6 在甜品店,花束不在手上)
- [ ] 段 6→7:**场景切换** 甜品店 → 卧室,需要花束带回并插入花瓶
- [ ] 段 7→8:光线从**午后暖黄**渐变到**夕阳橘粉**
- [ ] 段 8→9:温柚在窗边(段 8 末尾)→ 空镜余晖(段 9),衔接靠剪映转场
